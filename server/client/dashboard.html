About Fireball...
*Reverse-proxy WAF — sits in front of your app, intercepts every HTTP request before it reaches your code.
*Regex engine — pattern-matches against PATH, QUERY_STRING, HEADERS, and BODY zones using compiled regex rules.
*Behavioral/Volumetric Log Analyzer — runs every 60s, detects Brute Force, Scanners, DDoS, SQLi/XSS probing from log patterns.
*IF Anomaly Scanner — Isolation Forest unsupervised scan pe traficul stocat; identifica anomalii comportamentale pentru review manual.
*Zero-config database — DuckDB stores all request logs and firewall actions in a single embedded file (fireball.db).
